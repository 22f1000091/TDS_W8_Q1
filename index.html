Sample Earnings Calculation (Code)


def earnings_per_share(net_income, preferred_dividends, weighted_avg_shares):
    """
    Basic EPS calculation used in quarterly reporting.
    """
    return (net_income - preferred_dividends) / weighted_avg_shares

# Example:
net_income = 12_500_000      # in currency units
preferred_dividends = 500_000
weighted_avg_shares = 5_000_000

eps = earnings_per_share(net_income, preferred_dividends, weighted_avg_shares)
print(f"EPS for the quarter: {eps:.2f}")

Financial Formulas (MathJax)

Net Present Value (NPV):

$$ \text{NPV} = \sum_{t=1}^{T} \frac{CF_t}{(1 + r)^t} $$

Earnings Per Share (EPS):

$$ \text{EPS} = \frac{\text{Net Income} - \text{Preferred Dividends}}{\text{Weighted Avg. Shares}} $$

Return on Equity (RoE):

$$ \text{RoE} = \frac{\text{Net Income}}{\text{Average Shareholders' Equity}} $$
